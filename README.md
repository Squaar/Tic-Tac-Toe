Tic-Tac-Toe
================
Matt Dumford - mdumfo2@uic.edu


##To Run
- Use `ant` to build.
	- If ant complains about a missing lib directory just `mkdir lib`.
- `java -jar mdumfo2_TicTacToe.jar BOARD`
	- ex. `java -jar mdumfo2_TicTacToe.jar b b b X b O O X b`
	- X for player X
	- O for player O
	- b for blank spaces
